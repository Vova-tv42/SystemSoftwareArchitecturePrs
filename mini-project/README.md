# Безпечне копіювання файлів

## Призначення проєкту

Це мінімальна консольна програма, яка копіює вміст одного файлу в інший. Програма використовує низькорівневі системні виклики Linux і не перезаписує файл призначення, якщо він вже існує.

## Використані системні виклики

- `open()` - відкриття вхідного файлу та створення вихідного файлу.
- `read()` - читання даних з вхідного файлу.
- `write()` - запис даних у вихідний файл.
- `close()` - закриття файлових дескрипторів.

## Компіляція

```bash
./build.sh
```

## Запуск

```bash
./scopy <source_file> <destination_file>
```

## Приклади

Створити тестовий файл:

```bash
echo "Test input 1" > input.txt
```

Скопіювати файл:

```bash
./scopy input.txt output.txt
```

Перевірити результат:

```bash
cat output.txt
```

Спроба скопіювати у файл, який вже існує:

```bash
./scopy input.txt main.c
```

У цьому випадку програма виведе помилку і не перезапише `output.txt`.

## Як працює програма

Програма перевіряє, що користувач передав рівно два аргументи: шлях до вхідного файлу і шлях до файлу призначення. Далі вона відкриває вхідний файл для читання та створює файл призначення з прапорцем `O_EXCL`, щоб не перезаписати наявний файл.

Після цього програма читає дані з вхідного файлу частинами у буфер і записує ці частини у файл призначення. Якщо під час відкриття, читання, запису або закриття файлів виникає помилка, програма виводить коротке повідомлення через `perror()` і завершується з ненульовим кодом.
