# Практична робота №11: Дослідження та розробка демонів для моніторингу сигналів та аналізу енергетичних мікромереж

## Налаштування та запуск

**Компіляція програми:**

Для компіляції використовується скрипт `build.sh`.
```
./build.sh
```

## Використання файлів програм та пояснення

### task1.c
Програма працює у фоновому режимі та відстежує сигнали, надіслані їй. Для кожного отриманого сигналу фіксується PID, UID відправника та ім'я процесу, що надіслав сигнал. Дані записуються у файл `/tmp/signals.log`.
