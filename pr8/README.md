# Практична робота №8: Системні виклики в UNIX/POSIX (файлові операції, fork(), qsort(), write(), read(), lseek() тощо

## Налаштування та запуск

**Компіляція всіх завдань:**

Для компіляції програми використовується скрипт build.sh.
```
./build.sh
```
