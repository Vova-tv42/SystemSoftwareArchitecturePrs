# Практична робота №9

## Налаштування та запуск

**Компіляція всіх завдань:**

Для компіляції програми використовується скрипт build.sh.
```
./build.sh
```

## Використання файлів програм та пояснення

### task1.c
Програма читає дані з `/etc/passwd` за допомогою `getent passwd`, виділяє UID користувачів та виводить список тих, чий UID більше або дорівнює 1000.
